# Contact Microsoft Support to Enable SMTP

## If You Want to Stick with Microsoft 365

Since we can't find the SMTP auth setting in the admin center, you'll need Microsoft's help.

### Option 1: Submit a Support Request

1. Go to: https://admin.microsoft.com
2. Click "Support" in the bottom left
3. Click "New service request"
4. Describe your issue:

> "I need to enable SMTP AUTH for my Microsoft 365 tenant. I need to use programmatic email sending for my application. I'm getting this error: 'SmtpClientAuthentication is disabled for the Tenant.' Please enable SMTP authentication for my user: info@projectplanner.us"

### Option 2: Call Microsoft Support

1. Go to: https://admin.microsoft.com
2. Click "Support" → "Call support"
3. Tell them: "I need SMTP authentication enabled for my tenant"

They can enable it remotely in about 5 minutes.
